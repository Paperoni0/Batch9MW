# Batch9MW
## About

Batch9MW is a collaborative project created by students of SMA Maitreyawira Palembang Batch 9. Originally developed as a school assignment, it has grown into a digital archive dedicated to preserving creative works and showcasing the achievements of our graduating batch.

## Tech Stack
- HTML
- CSS
- JavaScript

## Features
- Poetry Collection
- Project Archive
- Responsive Design
